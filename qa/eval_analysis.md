# 新能源知识图谱问答系统对比评估分析总结

> 本报告基于 `qa/evaluate.py` 在 **Neo4j 已连接、LLM 已配置** 环境下的运行结果，对纯 LLM 与 KG+RAG 两种模式进行系统性对比分析。测试集已扩展至 **60 道问题**（原 50 题 + 新增 10 道边界题），并新增**准确率、召回率**等核心指标。

## 二、核心对比表格

### 2.1 汇总指标

| 指标 | 纯 LLM | KG+RAG | 差异 |
|------|--------|--------|------|
| 意图准确率 | 98.3% | 98.3% | — |
| 实体命中率 | 93.3% | 93.3% | — |
| 实体识别精确率 | 91.7% | 91.7% | — |
| 实体识别召回率 | 92.3% | 92.3% | — |

| **答案质量（LLM 判分 /100）** | **83.9** | **82.0** | **-1.8** |

| 本地兜底次数 | 0 | 0 | — |

### 2.2 指标说明

1. **答案准确率（Answer Accuracy，关键词命中率）**
   - 含义：每道题目预设若干期望答案关键词，计算这些关键词在系统最终回答中出现的比例，再对所有题目取平均。
   - 作用：量化回答与标准事实的贴合程度，抑制 LLM 自由发挥导致的"幻觉"。
   - 局限：只看关键词是否出现，不判断语义完整性，容易高估部分正确或泛泛而谈的回答。

2. **答案质量（LLM-as-judge Score）**
   - 含义：使用另一个 LLM 作为裁判，从相关性、完整性、事实正确性、表达清晰度四个维度对回答进行 0-100 评分。
   - 作用：弥补关键词匹配的不足，对同义改写、部分遗漏、逻辑混乱等情况更敏感。
   - 局限：裁判模型本身存在主观性和宽松度；使用与生成模型相同的 qwen-turbo 时，可能存在一定偏向性。

3. **实体识别精确率 / 召回率 / F1（Entity Precision / Recall / F1）**
   - 精确率 = 预测实体中属于期望实体的比例。
   - 召回率 = 期望实体中被成功预测的比例。
   - F1 = 精确率与召回率的调和平均。
   - 作用：比单一的"实体命中率"更全面地评价实体抽取能力。

4. **KG 召回率（KG Recall）**
   - 含义：对于关系类、列表类问题，计算期望实体在知识图谱检索结果中的覆盖比例。
   - 作用：直接反映知识图谱数据完整性与检索策略的有效性。

## 三、关键发现

### 3.1 加入边界题后，指标更真实

相较于原先 50 题全部"理想问法"的测试集，加入 10 道边界题后：

- **实体识别精确率从 100% 降至 91.7%**
- **实体识别召回率从 100% 降至 92.3%**
- **意图准确率从 100% 降至 98.3%**
- **KG 召回率从 100% 降至 97.0%**

这说明边界题有效暴露了系统在别名、口语化、多实体、错别字等场景下的薄弱环节，评估结果更具区分度。

### 3.2 知识图谱仍显著提升关键词命中率

- **KG+RAG 关键词命中率达到 80.1%**，较纯 LLM 的 69.0% **提升 11.1 个百分点**。
- 在事实密集型关系查询上（企业、材料、应用、政策），KG+RAG 优势明显。
- 典型提升案例：

| 问题 | 纯 LLM 关键词准确率 | KG+RAG 关键词准确率 | 提升 |
|------|---------------------|---------------------|------|
| 纯电动汽车是什么？ | 25% | 100% | **+75%** |
| 磷酸铁锂电池有哪些竞争技术？ | 100% | 100% | — |
| 质子交换膜燃料电池有哪些材料？ | 50% | 100% | **+50%** |
| 分布式光伏发电有哪些政策支持？ | 25% | 100% | **+75%** |
| 铁铬液流电池有哪些材料？ | 0% | 100% | **+100%** |
| 风电制氢有什么应用？ | 33% | 100% | **+67%** |

### 3.3 LLM-as-judge 显示语义质量接近

| 指标 | 纯 LLM | KG+RAG | 差异 |
|------|--------|--------|------|
| 答案准确率（关键词） | 69.0% | 80.1% | +11.1% |
| 答案质量（LLM 判分） | 83.9 | 82.0 | -1.8 |

关键词准确率显示 KG+RAG 优于纯 LLM（+11.1%），但使用更严格的 qwen-plus 判分后，KG+RAG 平均分反而略低于纯 LLM（82.0 vs 83.9）。这说明：
- qwen-plus 作为 judge 比 qwen-turbo 更严格，能识别出 KG+RAG 回答中更简短、覆盖局部的问题；
- 纯 LLM 虽然 keyword 命中少，但回答更完整、解释更充分，语义质量未必差；
- KG+RAG 的优势主要体现在**事实准确性**和**关键词命中**上，但在回答的完整性和自然度上仍有提升空间。

## 四、新增 10 道边界题表现分析

新增题目覆盖 **别名识别、多实体比较、复合关系、口语化、错别字、未知技术** 等场景：

| 问题 | 类型 | KG+RAG 实体识别 | KG+RAG 关键词准确率 | 说明 |
|------|------|----------------|---------------------|------|
| 锂电池有什么优缺点？ | 别名 | ✅ 磷酸铁锂电池 | 80% | 别名"锂电池"正确映射 |
| 三元电池和铁锂电池哪个好？ | 别名+多实体 | ✅ 两个都识别 | 60% | 别名同时生效 |
| 钠电池有哪些材料？ | 别名 | ❌ 未识别 | 40% | jieba 未切出"钠电池"，同义词未触发 |
| 燃料电池汽车有哪些应用场景？ | 别名/完整实体冲突 | ❌ 识别为质子交换膜燃料电池 | 60% | synonym 把"燃料电池"提前映射，完整实体被拆分 |
| 磷酸铁锂电池有哪些材料和公司？ | 复合关系 | ✅ 正确 | 100% | 多关系查询处理较好 |
| 纯电动汽车和氢燃料电池汽车哪个好？ | 多实体 | ⚠️ 多识别出质子交换膜燃料电池 | 83% | 完整实体识别受 synonym 干扰 |
| 电动车的工作原理是什么？ | 口语化/不完整 | ✅ 纯电动汽车 | 80% | 口语化表达能正确识别 |
| 光fu逆变器有什么作用？ | 错别字 | ⚠️ 实体识别正确，但意图误判为 list | 60% | fuzzy 匹配生效，但"作用"触发 list 意图 |
| 固态电池有哪些优点？ | 未知技术 | ❌ 未识别 | 67% | 图谱中无"固态电池"，LLM 基于知识作答 |
| 钒电池的循环寿命是多少？ | 别名 | ❌ 未识别 | 25% | jieba 未切出"钒电池"，同义词未触发 |

### 4.1 暴露的主要问题

1. **同义词依赖 jieba 分词**
   - "钠电池"、"钒电池"等同义词在 jieba 分词中未被切分为独立 token，导致 synonym 映射失效。
   - 改进方向：增加基于子串的同义词匹配，或添加 jieba 自定义词典。

2. **完整实体与 synonym 冲突**
   - "燃料电池汽车"应被识别为完整技术实体，但"燃料电池" synonym 优先映射到"质子交换膜燃料电池"。
   - 改进方向：优先匹配更长的完整实体名，再触发同义词映射。

3. **意图分类对短词/错别字敏感**
   - "光fu逆变器有什么作用"被误判为 list 意图，因"有什么"被 list 关键词命中。
   - 改进方向：在意意图分类时结合实体识别结果，若已识别实体则优先判为 property/relation。

4. **未知技术缺乏明确拒答**
   - "固态电池"未在图谱中，系统未明确告知用户"未收录"，而是直接由 LLM 作答。
   - 改进方向：增加未知实体检测与拒答机制。

## 六、总结与建议

### 6.1 当前系统优势

- **KG+RAG 在关键词命中率上显著优于纯 LLM**（+11.1%），说明知识图谱有效减少了事实型问题的幻觉。
- **响应时间与纯 LLM 基本持平**（约 1180 ms），KG 检索未带来明显延迟。
- **标准问法下的实体识别和意图分类表现稳定**。

### 6.2 待改进方向

1. **增强别名/同义词识别鲁棒性**
   - 不依赖 jieba 分词结果，增加子串级同义词匹配。
   - 优先识别完整实体名，避免"燃料电池汽车"被拆分为"燃料电池"+"汽车"。

2. **提升意图分类的上下文感知能力**
   - 当问题中已识别到实体时，避免"有什么"等泛化词将 property/relation 问题误判为 list。

3. **增加未知技术拒答机制**
   - 对于图谱中未收录的技术，明确返回"当前知识图谱暂未收录"，而非直接由 LLM 编造。

4. **扩展测试集**
   - 继续增加边界 case（如多轮对话、跨领域关联查询），进一步拉开系统能力的区分度。
